# command-ts

A monorepo for handling discord.js commands with TypeScript.

## 📦 Packages

This monorepo contains the following packages:

-   **[command-ts](./packages/command-ts)** - Core package for handling Discord.js commands, events, buttons, modals, and select menus with TypeScript
-   **[template](./packages/template)** - Template package for the monorepo

## 🚀 Getting Started

### Prerequisites

-   Node.js 22.0.0 or later
-   pnpm 10.27.0 or later

### Installation

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Run in development mode
pnpm dev
```

## 📜 Available Scripts

-   `pnpm build` - Build all packages in the monorepo
-   `pnpm dev` - Run all packages in development mode with watch
-   `pnpm test` - Run tests for all packages
-   `pnpm lint` - Check code formatting
-   `pnpm fmt` - Format code
