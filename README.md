A package for handling discord.js commands with TypeScript

## Installation

To install command-ts, run one of the following commands based on your preferred package manager:

#### NPM

```sh
npm install command-ts
```

#### PNPM

```sh
pnpm add command-ts
```

#### Yarn

```sh
yarn add command-ts
```
